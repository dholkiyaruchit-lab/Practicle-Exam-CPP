This repository contains 5 basic Object-Oriented Programming (OOP) programs written in C++.
The programs demonstrate important OOP concepts such as:

Classes & Objects
Constructors
Encapsulation
Inheritance
Polymorphism
Abstract Classes
Virtual Functions
📂 Programs Included

Q1 - Laptop Information System

🔹 Concepts Used
Class & Object
Constructor
Encapsulation
🔹 Description

This program creates a Laptop class with:

Laptop Name
Price
Processor

It displays laptop details using objects.

Q2 - Bank Account Management System

🔹 Concepts Used
Constructor
Encapsulation
Member Functions
🔹 Description

This program simulates a simple bank account system with:

Credit Amount
Debit Amount
Balance Display

It validates balance before withdrawal.

Q3 - Shape Area Calculation using Inheritance

🔹 Concepts Used
Inheritance
Getter & Setter Functions
Area Calculation
🔹 Description

This program creates:

Shape Base Class
Circle Derived Class
Rectangle Derived Class

It calculates and displays area of shapes.

Q4 - Runtime Polymorphism using Virtual Function

🔹 Concepts Used
Virtual Functions
Function Overriding
Runtime Polymorphism
Dynamic Memory Allocation
🔹 Description

This program demonstrates runtime polymorphism using:

Base class pointer
Virtual display_Details() function

Both Circle and Rectangle override the base class function.

Q5 - Abstract Class Example

🔹 Concepts Used
Pure Virtual Function
Abstract Class
Polymorphism
🔹 Description

This program creates:

Abstract class Vehicle
Derived classes Car and Bike

Both classes implement:

startEngine()
vehicledrive()

📚 Learning Outcomes

After completing these programs, you will understand:

How classes and objects work
Importance of constructors
Inheritance in C++
Runtime polymorphism
Abstract classes and virtual functions
